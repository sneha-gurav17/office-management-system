# Office Management System 

A full-stack **Office Management System** designed to manage employees, attendance, payroll, and personal details efficiently.

---

## 🛠️ Tech Stack

### 🔹 Backend
- Java
- Spring Boot
- Hibernate (JPA)
- MySQL

### 🔹 Frontend
- React.js (Vite)
- CSS (Custom UI)

### 🔹 Tools
- Postman

---

## Features

### Employee Management
- Add Employee
- View Employees
- Update & Delete Employees
- Search by Name, Role, Department

---

### Employee Details
- Contact Information
- Address Details
- Aadhaar & PAN
- Bank Details
- Secure data handling

---

### Attendance Management
- Manual attendance entry (HR-based system)
- Date-wise tracking
- Check-in / Check-out
- Total working hours
- Status (Present / Absent)

---

### Payroll Management
- Generate salary
- Bonus & deductions
- Net salary calculation
- Mark salary as Paid

---

### Dashboard
- Employee count
- Attendance overview
- Payroll summary
- Selected employee details

---

## Key Concepts Used
- REST API Development
- MVC Architecture
- JPA Relationships (One-to-One, One-to-Many)
- API Integration
